## 의존성 주입 유형 🔄

Spring에서는 다음과 같은 다양한 의존성 주입 방식을 제공합니다:

1. 생성자 주입 (Constructor Injection)
2. 수정자 주입 (Setter Injection)
3. 필드 주입 (Field Injection)
4. 메서드 주입 (Method Injection)

각 방식의 장단점과 적절한 사용 시점에 대해 이 섹션에서 다룹니다.
