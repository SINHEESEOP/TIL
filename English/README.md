# 🌏 English Study

영어 실력 향상을 위한 체계적인 학습 공간입니다.

## 🎯 Why Here?
프로게이머 연습생 시절부터 프로 생활까지, 게임에 몰두하느라 기초 영어 공부를 제대로 하지 못했습니다. 이제는 개발자로서 성장하기 위해, 그리고 더 넓은 세계와 소통하기 위해 기초부터 차근차근 영어 실력을 쌓아가려고 합니다.
